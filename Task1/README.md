# AI Learn Hub - Landing Page

## Oasis Infobyte Internship (OIBSIP)

### Task 1 - Landing Page

## 📌 Project Overview

AI Learn Hub is a responsive landing page developed as part of the Oasis Infobyte Web Development Internship. The website promotes an online learning platform offering courses in Artificial Intelligence, Web Development, Data Structures, and other modern technologies.

The project demonstrates fundamental web development concepts using HTML, CSS, and JavaScript while focusing on responsive design, user-friendly layout, and modern UI elements.

---

## 🚀 Features

- Responsive Navigation Bar
- Hero Section
- Feature Cards
- Popular Courses Section
- Statistics Section
- Student Testimonials
- Call-to-Action Section
- Footer with Social Media Icons
- Smooth Hover Effects
- Sticky Navigation Bar
- Mobile Responsive Design
- Clean and Modern User Interface

---

## 🛠 Technologies Used
