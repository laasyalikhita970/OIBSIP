# 🌐 Oasis Infobyte Web Development & Designing Internship (Level 1)

This repository contains the projects completed as part of the **Oasis Infobyte Web Development & Designing Internship – Level 1**.

The internship focuses on building a strong foundation in front-end web development using **HTML, CSS, and JavaScript** through hands-on projects.

---

## 📌 Level 1 Tasks

### ✅ Task 1 – Landing Page

A modern and responsive landing page designed using HTML, CSS, and JavaScript.

**Features**
- Responsive design
- Modern UI
- Navigation bar
- Hero section
- Call-to-action buttons
- Footer
- Clean and organized code

**Technologies Used**
- HTML5
- CSS3
- JavaScript

📁 Folder: `Task1`

---

### ✅ Task 2 – Personal Portfolio

A personal portfolio website showcasing profile, skills, projects, education, and contact details.

**Features**
- Responsive layout
- About section
- Skills section
- Projects section
- Education section
- Resume download option
- Contact section

**Technologies Used**
- HTML5
- CSS3
- JavaScript

📁 Folder: `Task2`

---

### ✅ Task 3 – Temperature Converter

A web application that converts temperatures between Celsius, Fahrenheit, and Kelvin.

**Features**
- Celsius ↔ Fahrenheit conversion
- Celsius ↔ Kelvin conversion
- Fahrenheit ↔ Kelvin conversion
- Input validation
- Reset functionality
- Responsive user interface

**Technologies Used**
- HTML5
- CSS3
- JavaScript

📁 Folder: `Task3`
