# Temperature Converter

## Oasis Infobyte Internship - Task 3

### 📌 Project Overview

This project is a responsive Temperature Converter developed using HTML, CSS and JavaScript.

It converts temperatures between:

- Celsius (°C)
- Fahrenheit (°F)
- Kelvin (K)

The application validates user input and instantly displays the converted temperature.

---

## 🚀 Features

- Celsius ↔ Fahrenheit
- Celsius ↔ Kelvin
- Fahrenheit ↔ Kelvin
- Input Validation
- Reset Button
- Responsive Design
- Modern UI
- Beginner-Friendly Code

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
